# Repo Template Review

Review changes in this repository against these questions:

1. Does every changed template remain standalone and copy-ready?
2. Do related `AGENTS.md`, `CLAUDE.md`, and `GEMINI.md` files still agree on the core rules?
3. Were commands, starter manifests, and docs kept aligned?
4. Did the change leave stale placeholder text, misleading examples, or incomplete copy-after-edit guidance?
5. Did the change make the repository easier or harder to maintain as a template library?
