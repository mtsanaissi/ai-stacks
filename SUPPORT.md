# Support

This repository is maintained in the open and kept free to use. If you need help or want to support ongoing maintenance, use the paths below.

## Current Support Paths

- Bug reports and actionable repo problems: open an issue in this repository
- Contribution and change-process questions: see [CONTRIBUTING.md](CONTRIBUTING.md)
- General repository support and maintenance updates: use this guide as the current source of truth

## Bug Reports

Use GitHub Issues for bugs, regressions, broken docs, template drift, or workflow problems.

When opening an issue, include:

- what you expected to happen
- what actually happened
- the files or paths involved
- reproduction steps, if applicable

Issue link:

- [Open an issue](https://github.com/mtsanaissi/ai-stacks/issues)

## Sponsorship And Donations

If this repo saves you time or you want to support further work on templates, skills, and repo automation:

- [GitHub Sponsors](https://github.com/sponsors/mtsanaissi)
- [Buy Me a Coffee](https://buymeacoffee.com/mtsanaissi)

Financial support helps sustain maintenance and improvements, but it does not gate access to the repository or its contents.
