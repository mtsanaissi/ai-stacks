# Report Design

Use this file to keep report UX conventions explicit after copying the template.

Recommended sections:

- Page types and navigation patterns
- Filter and slicer behavior
- Drillthrough, tooltip, and bookmark rules
- Accessibility and mobile-layout expectations
- Export and subscription considerations

Conventions:

- Keep navigation and filtering predictable and easy to explain to users.
- Preserve accessibility, contrast, labeling quality, and mobile usability where the report supports it.
- Use consistent page structure for overview, detail, and drillthrough flows.
- Review export, print, and Analyze in Excel impact when visuals or filters change.
- Keep hidden pages, tooltip pages, and bookmark states documented if they affect user behavior or security assumptions.
