# Avoid Full Project File Paths Unless Essential

## Summary

Do not use full file paths when referencing project files unless the task requires them to accomplish the goal.

## Context

The user asked for this to be stored as durable guidance for future sessions.

## Remember

Reference project files without full absolute paths by default. Use full file paths only when they are essential to complete the task or make the result work.
